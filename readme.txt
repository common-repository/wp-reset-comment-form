WP comment form reset after submit.

== Description ==

This will reset comment formfields after submit. Purpose of this plugin is it'll prevent comment data to store in COOKIES. Easy to use.

= Features include: =

* Purpose of this plugin is it'll prevent comment data to store in COOKIES. 
* Easy to use

== Installation ==

1. Install WP Reset Comment Form either via the WordPress.org plugin directory, or by uploading the files to your server
2. Activate the plugin through the 'Plugins' menu in WordPress
